A feature-rich recipe management app built with Node.js, Express, MongoDB, and EJS, allowing users to perform CRUD operations on recipes. Additionally, it includes a custom meal planner that suggests meals based on the user's health conditions.
User Authentication – Secure login & signup using sessions
Recipe Management (CRUD) – Add, update, delete, and view recipes
Image Upload – Store recipe images in MongoDB
Search & Filter – Find recipes by ingredients, cuisine, or category
Custom Meal Planner – Suggests meals based on user health conditions
Interactive UI – Built with EJS & Bootstrap for a clean experience
